# Volumetric Blob

An animated, translucent blob with swirling internal caustics — inspired by organic plasma/gel spheres. Renders in **purple** or **silver**.

## Run

Open `index.html` in a modern browser, or serve locally:

```bash
npx serve .
```

Then visit `http://localhost:3000`.

## Controls

- **Drag** — rotate the blob
- **Purple / Silver buttons** — switch palette
- **P** or **1** — purple
- **S** or **2** — silver

Built with Three.js, GLSL raymarching, and bloom post-processing.
